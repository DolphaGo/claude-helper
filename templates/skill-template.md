# Skill 이름

## 목적
이 skill이 해결하는 문제나 제공하는 기능을 설명합니다.

## 사용 시점
- 언제 이 skill을 사용해야 하는지
- 어떤 상황에서 유용한지
- 예: "사용자가 '코드 리뷰 해줘'라고 요청할 때"

## 실행 단계

### 1단계: 준비
- 필요한 파일이나 정보 수집
- 사전 조건 확인

### 2단계: 핵심 작업
- 주요 로직 실행
- 도구 사용 (Read, Grep, Bash 등)

### 3단계: 결과 정리
- 결과 포맷팅
- 사용자에게 리포트

## 출력 형식
```
예상되는 출력 형식을 보여줍니다.
```

## 예제

### 입력
```
사용자: [예제 요청]
```

### 출력
```
assistant: [예상 응답]
```

## 주의사항
- 특별히 주의할 점
- 제약사항이나 한계
- 에러 처리 방법

- `Read`: 파일 읽기
- `Grep`: 코드 검색
- `Bash`: 명령 실행
